First Graphic

Load the ggplot2 library

library(ggplot2)

Letâ€™s start by creating a bar chart of counts of species within the database.

ggplot(data=scdb) + geom_bar(mapping = aes(x=spp1))

Letâ€™s make it more readable by limiting the chart to spruce only:

ggplot(data=dplyr::filter(scdb, spp1=='SS'|spp1=='NS')) + geom_bar(mapping = aes(x=spp1))

Letâ€™s extend this idea by creating a bar chart using two variables: Yield Class and Species.

ggplot(data=scdb, aes(yc1, fill=spp1)) +  geom_histogram()
